\begin{abstract}
This draft develops a no-new-data reassessment of \textcite{bresnan2007}. The
current analyses reproduce the production-choice target from
\texttt{languageR::dative}, add explicit null and partial-pooling checks, and
test whether the resulting conditioning profile transports to the Spoken
BNC2014 dative dataset. The working result is that modern modelling preserves
the classic production conclusion, while BNC2014 provides positive but partial
cross-corpus transport evidence. The manuscript uses that result to keep apart
production probability, acceptability preference, and grammatical possibility.
\end{abstract}

\noindent\textbf{Keywords:} dative alternation; probabilistic grammar; grammaticality; corpus linguistics; acceptability
