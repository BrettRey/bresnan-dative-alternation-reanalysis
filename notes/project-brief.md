# Project Brief

## Seed

The project was set up from the revised recommendation that Bresnan, Cueni,
Nikitina, and Baayen's "Predicting the dative alternation" is the strongest
no-new-data empirical target. Bard et al. remains the best conceptual frame for
acceptability measurement, but its original participant-level data situation is
weaker. Bresnan et al. has an open-data path through `languageR::dative`, with
possible generalization datasets in BNC2014 and DAIS.

## Core Question

What does a probabilistic model of dative-alternation production tell us about
grammatical possibility?

## Initial Contribution Shape

1. Reproduce the original-style dative analysis.
2. Update the modelling and uncertainty story.
3. Test whether the conditioning profile travels across later datasets.
4. Use the case to distinguish rare, dispreferred, conditioned, and
   ungrammatical patterns.

## Relation To Brett's Program

This paper should feed *(Un)grammatical* and the projectibility-profile work. A
construction's production profile can be evidence about grammar, but only under
declared scope conditions. A form's rarity is not the same as ungrammaticality,
and a probabilistic alternation profile is not merely a frequency table.
